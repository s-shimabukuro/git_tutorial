terminal コマンド 一覧

空ファイルの作成

touch ファイル名

git コマンド 一覧

git commit -v

git push -u origin main

リモートリポジトリ

対応するリモートを表示

git remote

対応するリモートのURLを表示

git remote -v

リモートの情報の詳細

git remote show リモート名

リモートリポジトリの追加

git remote リモート名 リモートリポジトリのURL

ローカルのブランチをリモートリポジトリにpush、デフォルトのpush/pull先に設定

git push -u リモート名 プッシュするブランチの名前

リモートリポジトリの内容をローカルリポジトリに反映

git pull リモート名　プルするブランチ名

リモート名の変更

git remote rename 旧リモート名 新リモート名

リモートリポジトリの削除

git remote rm リモート名

取り消し

ワークツリーの変更を破棄

git restore ファイル名

ステージの変更を破棄

git restore --sataged ファイル名

ワークツリーとステージの変更を破棄

git resotre --staged --worktree ファイル名

直前のコミットをやり直す

git commit --amend

エイリアス(別名)をつける

commit → ci

git config --global alias.ci commit

status → st

git config --global alias.st status

branch → br

git config --global alias.br branch

checkout → co

git config --global alias.co checkout
